## Usage

To use this template, follow these steps:

1. **Clone the Repository**: 
    ```bash
    git clone https://github.com/Rahul-Mehra00/rahulmehra.github.io.git
    ```
2. **Customization**:
   - Update `index.html` with your information.
   - Modify `style.css` for custom styles.
